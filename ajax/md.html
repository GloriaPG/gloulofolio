manoderecha

manoderecha is a project management system. My position on this project was as a Technical Team Leader.

  • manoderecha web
  • manoderecha system

My activities here were:

  • - Project Management.
  • - Decision making.
  • - Deploying the application in production and staging.
  • - Developer.
  • - Quality Manager.
  • - Improvement implementation.
  • - Server maintenance.

My achievements in this job were:

  • - Implementing Continuous Integration with Jenkins.
  • - Microservices architecture design.
  • - Fixed bugs.
  • - Promote a new design for the application.
  • - Starting the new application manoderecha in node.js
  • - Acquire knowledge infrastructure.
  • - Acquire knowledge for application architecture.
  • - Acquire knowledge for application design.

The tools I used in this project were:

  • - PHP.
  • - MySql.
  • - Angular, Bootstrap, Html, JS.
  • - DBFinder and Propel.
  • - Vagrant.
  • - GitHub.
  • - Centos.
  • - Mandrill.
  • - WordPress.
  • - Node.js, Express, Kraken.
  • - Symfony 1.2
