My job within this company was:

  • App Development with Android 2.3. This app was connected to an ERP app (Open ERP), using a REST develop in Java, and RPC.

  • Development of modules for OpenERP, using python and manager PostgreSQL database.

  • Creating reports and charts in Pentaho, using data cubes and MDX queries.

  • Management ERP OpenERP.

My accomplishments within the company were to create a methodology for agile development, this methodology was based on the ASD methodology and ITIL principles. This methodology allowed the developer to generate a knowledge base, with each error, with each job and each activity, this allows new developers to support materials and equipment which could work quickly.
